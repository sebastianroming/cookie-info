Ein simples Plugin zur Konformität mit der EU Cookie Richtline

== Description ==

Gemäß EU-Richtlinie sind Webseitenbetreiber dazu verpflichtet, Ihre Besucher auf die Benutzung von Cookies hinzuweisen.
Dieses Plugin blendet einmalig beim ersten Besuch eine Meldung mit frei definierbarem Text ein.

Achtung: Es wird keine Gewähr für Konformität mit den Gesetzen / Regelungen übernommen!

== Installation ==

Installation via [WordPress Pluginverzeichnis](https://wordpress.org/plugins/cookie-info/).
